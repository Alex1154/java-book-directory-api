# Book Directory API with JAVA

## Esse projeto deve conter uma API RESTful para listagem, criação, busca, modificação e exclusão de livros.

## Características que deve conter na API:

- Persistência de dados
- Métodos HTTP (GET, POST, PUT, DELETE)
- Não pode Haver livros repetidos
- Livros devem conter Id, Nome, Autor e a Edição.
